## Para compilar el proyecto

- [se requiere LARAVEL 9).
- [MINIMO PHP 8.0).
- [Aplicar los siguientes comandos:)
- [composer install)
- [npm install)
- [Renombrar el archivo .env.example por .env)
- [configurar la base de datos en local mySQL)
- [php artisan migrate)
